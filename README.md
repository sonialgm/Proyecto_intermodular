# Proyecto intermodular - Conversaciones Anónimas

Este proyecto es una plataforma sencilla desarrollada en **PHP + MySQL** que permite a los usuarios iniciar y participar en conversaciones forma anónima.

## Funcionalidades principales

### Usuarios 
- Registro de nuevas conversaciones (`register.php`)
- Inicio de sesión en una conversación (`login.php`)
- Cierre de sesión (`logout.php`)
- Visualización y gestión del perfil (`perfil.php`)

### Administrador
- Login para acceder como administrador (`login_admin.php`)
- Cierre de sesión del admininstrador (`logout_admin.php`)
- Script para crear administradores manualmente al inicio (`insertar_admin.php`)

## Tecnologías utilizadas
- **PHP** 
- **MySQL**
- **HTML + CSS**
- Scripts de prueba:
  - `.sh` para testear desde terminal.
  - `.js` para pruebas de API cliente.
